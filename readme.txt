Plugin to connect your WordPress website with the Wiwi Websolutions Bookingsystem.

== Description ==

Plugin that is being used by Wiwi Websolutions to manage your site.

== Installation ==

1. Upload the plugin package to the plugins directory.
2. Activate the plugin.
3. Done!
